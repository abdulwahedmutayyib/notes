Notes App
Description
The Notes App is a simple Python application that allows users to create, view, update, and delete notes. It provides a user-friendly interface with different themes of colors to choose from. The application integrates with a SQLite database to store notes, making it easy to manage and retrieve notes across sessions.

Features
Create new notes
View existing notes
Update notes
Delete notes
Choose from different themes of colors
Requirements
Python 3.x
Tkinter (Python GUI library)
SQLite (included in Python's standard library)

Usage
Upon running the application, the main window will open, displaying existing notes (if any).
To create a new note, click on the "New Note" button.
To view or edit an existing note, click on the note in the list.
To delete a note, select the note and click on the "Delete" button.
Choose a theme from the options provided to change the color scheme of the application.
